Website ini dirancang untuk membantu memahami konsep dan simulasi berbagai teknik dalam Blockchain Technology. Ikuti langkah-langkah berikut:

  • Klik pada bagian Menu Bar di bagian atas untuk memilih simulasi atau fitur yang ingin dipelajari.

Penjelasan Menu Bar:

  1. Home: Tampilan awal Website yang berisi panduan dan teori dasar.
  2. About Me: Profil pembuat Website, CV, dan kontak.
  3. Hash: Masukkan input teks pada area Text Input, lalu hasil SHA-256 Hash akan muncul otomatis.
  4. Block: Simulasi struktur blok tunggal. Anda dapat mengganti nomor blok, data, dan melakukan mining (Proof of Work).
  5. Block Chain: Simulasi rantai blok. Pastikan Previous Hash blok baru sesuai dengan hash blok sebelumnya.
  6. ECC Signature: Simulasi tanda tangan digital (Digital Signature) menggunakan algoritma Elliptic Curve.
  7. Konsensus: Simulasi mekanisme konsensus terdistribusi antar node (User A, B, C).
