Dasar Teori Singkat

Blockchain adalah buku besar digital yang terdesentralisasi, didistribusikan, dan seringkali bersifat publik yang terdiri dari catatan-catatan yang disebut blok yang digunakan untuk mencatat transaksi di banyak komputer sehingga catatan tersebut tidak dapat diubah secara retroaktif tanpa mengubah semua blok berikutnya dan konsensus jaringan.

Penjelasan Menu Bar:

  • Home: Tampilan Awal Website.
  • About Me: Menampilkan profil pembuat website.
  • Hash: Masukkan teks ke dalam input. Hasil SHA-256 Hash akan otomatis muncul setiap kali teks diubah.
  • Block: Anda dapat mengubah nomor blok, memasukkan data, serta mengatur kecepatan mining. Tekan tombol Mine untuk memulai proses mining. Setelah hash dengan empat angka nol di awal ditemukan, nilai Nonce, Timestamp, dan Hash Block akan muncul.
  • Block Chain: Masukkan data, tekan Mine, lalu tekan Add Block untuk menambahkan blok baru. Perhatikan bahwa Previous Hash pada blok baru harus sama dengan hash blok sebelumnya.
  • ECC Signature: Klik Generate Key Pair untuk membuat Public Key dan Private Key. Tulis pesan, lalu tekan Sign Message untuk menghasilkan signature. Gunakan Verify Signature untuk mengecek validitas tanda tangan digital.
  • Konsensus: Simulasi dengan 3 pengguna (masing-masing memiliki saldo awal 100 coin). Buat transaksi, kirim, lalu tekan Mine Semua Transaksi untuk membuat blok baru. Jika data pada blok diubah secara manual lalu tombol Verify ditekan, blok tersebut akan ditandai merah muda (invalid). Tekan Consensus untuk memperbaiki data berdasarkan mekanisme konsensus.
